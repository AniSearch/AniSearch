# AniSearch
Search info on Anime and Manga via Discord.

# Config
Example:
```js
{ 
    "token": "xxx",
    "prefix": "!",
    "nsfw": "limited",
    "ownerID": "496477678103298052",
    "postgres": {
        "host": "1234",
        "port": 5432,
        "database": "anisearch",
        "user": "admin",
        "password": "1234"
    }
}
```
`prefix` and `nsfw` are the default config values for each guild
